# My Personal Website (Simple Template)

This is a lightweight, multi-page personal website template.

## Files
- `index.html` — Home page
- `blog.html` — Blog page with sample posts and an easy post template
- `about.html` — About / Contact page
- `style.css` — Styles (light colours)
- `images/placeholder.png` — Placeholder image (replace this with your logo/photo)

## How to use
1. Unzip the archive.
2. Open `index.html` in your browser.
3. Edit the HTML files with your text, replace `images/placeholder.png` with your image, and update contact links.
